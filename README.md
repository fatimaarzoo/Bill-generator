# Bill-Generator tool

A simple **dynamic Web Application** made using _Html/CSS_ And _javascript_ DOM for **real-time** calculations on screen for bill generation of tenents. And **simultaneously saving data** in the **databas**e using _Python _and _web-framework Flask._ It uses _SQLAlchemy_ with _sqlite_ **database** and stores the the whole bill with name and month and all entries and calculated. A **print** option is avalaible to print the table.

Just enter all the entries and and it will calculate the difference and amount in relatime. And for final amount click on the find total. Print and then click on save button or reset the bil.
